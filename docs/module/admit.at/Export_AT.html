Export_AT — Exports an Image_BDP to FITS.¶

This module defines the Export_AT class.

class admit.at.Export_AT.Export_AT(**keyval)[source]¶

Creates a FITS file from an image BDP.

Unlike the more automated export facility in ADMIT, this allows you to add a true FITS export atomically to the flow, with all of its flow dependencies.

At a future date this task will also export other types of BDPs, e.g. tables

Note that the output file is always overwritten.

Keywords

basename: string
Basename of the file. If left blank, it will be derived from the input BDP by replacing the extension (usually .im, .cim, .lim) with “fits”. This will thus also normally include whatever directory structure exists with the ADMIT tree. If the basename starts with “/” or ”./”, it is presumed to refer to an absolute reference, not something within the ADMIT directory. Default: “”.

Input BDPs

Image_BDP: count: 1
Input 2-D or 3-D image, such as output by Ingest_AT, LineCube_AT or Moment_AT.
